# Anchor Plan Builder

Веб-сервіс для побудови анкор-плану для закупок посилань на основі даних позицій та вигрузки з Ahrefs.

## Встановлення та запуск

### 1. Встановити залежності

```bash
pip install -r requirements.txt
```

### 2. Запустити бекенд

```bash
cd backend
python app.py
```

Сервер стартує на `http://localhost:5000`

### 3. Відкрити фронтенд

Відкрити файл `frontend/index.html` у браузері, або запустити через простий HTTP-сервер:

```bash
cd frontend
python -m http.server 8080
```

Потім відкрити `http://localhost:8080`

## Формат вхідних файлів

### Файл позицій (CSV/XLSX)

| URL | Keyword | 2025-01-01 | 2025-02-01 | 2025-03-01 |
|-----|---------|------------|------------|------------|
| https://site.com/page1 | купити ноутбук | 15 | 12 | 8 |
| https://site.com/page2 | ремонт телефону | 25 | 22 | 20 |

### Вигрузка Ahrefs (CSV/XLSX)

| Target URL | Anchor | Referring Page | Type | DR | Traffic |
|------------|--------|----------------|------|-----|---------|
| https://site.com/page1 | купити ноутбук | https://donor.com/page | dofollow | 45 | 150 |

## Можливості

- Автоматичний парсинг різних форматів (CSV, XLSX)
- Аналіз поточних позицій та динаміки
- Класифікація існуючих анкорів
- Визначення пріоритетів сторінок
- Генерація анкор-плану з безпечним розподілом
- Налаштування пропорцій, пріоритетів, кількості посилань
- Ручне редагування плану
- Експорт в XLSX
