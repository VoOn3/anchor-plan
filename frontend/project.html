Завантажте файл позицій та вигрузку беклінків (Ahrefs або Collaborator)

Вимоги до файлів

Реєстр позицій (CSV/XLSX)
  • Обов'язкові колонки: URL (або Page, Сторінка), Keyword (або Ключ, Query)
  • Колонки з датами (YYYY-MM-DD, DD.MM.YYYY) — позиція ключа на цю дату
  • Значення в колонках дат: число 1–1000 або порожнє
Приклад CSV 
URL,Keyword,2024-01-15,2024-02-15
https://example.com/page,ключове слово,12,8
https://example.com/blog,crm система,25,18
Вигрузка беклінків (CSV/XLSX)
  • Ahrefs: обов'язкова колонка Anchor (або Анкор). Опційно: Target URL, Referring Page, Type, DR, Traffic
  • Collaborator: XLSX з листом «Розподіл по операціях», колонки «Анкори» та «Цільова сторінка»
Приклад Ahrefs CSV 
Target URL,Anchor,Referring Page,Type
https://example.com/page,ключове слово,https://site.com/article,dofollow
https://example.com/page,тут,https://other.com/post,nofollow
